## Вікторина та питання вікторини на знання

Блок `quiz` відображає просте радіоопитування без оцінювання. Блок `question` відображає питання вікторини з самооцінюванням, правильною відповіддю, додатковим зворотним зв'язком для кожного вибору та налаштовуваною легендою.

---

### Вікторина

Блок «вікторина» — це просте опитування з радіокнопкою без правильної відповіді. Питання визначено у вступній частині YAML, а варіанти вибору використовують нотацію `( )`.

📖 [Специфікація Крамдауна — Вікторина](http://digital-docs.rpf-internal.org/docs/technology/codebases-and-products/raspberry-flavoured-markdown/specs/kramdown_rpf-legacy-spec#quiz)

```markdown
---
питання: Як ви ставитеся до цього проєкту на даний момент?
---

- ( ) Готові програмувати!
- ( ) Ще розігріваюся
- ( ) Трохи розгублені — але цікаві

```

---

## питання: Як ви ставитеся до цього проєкту на даний момент?

- () Готові до кодування!
- () Все ще розігрівається
- ( ) Трохи розгублений — але цікавий

---

### Завдання вікторини на знання — просте

Блок «питання» – це питання з самооцінюванням. У варіантах відповідей використовується символ `( )` для неправильних відповідей та `(x)` для правильних.

📖 [Специфікація Крамдауна — питання вікторини на знання](http://digital-docs.rpf-internal.org/docs/technology/codebases-and-products/raspberry-flavoured-markdown/specs/kramdown_rpf-legacy-spec#knowledge-quiz-question)

```markdown

Що повертає `sense.get_temperature()`?


- ( ) Рядок
- (x) Число з плаваючою комою
- ( ) Ціле число


```

Що повертає `sense.get_temperature()`?

- ( ) Рядок
- (x) Поплавок
- ( ) Ціле число

### Запитання вікторини на знання — із зворотним зв'язком для кожного варіанта

Окремі блоки «зворотного зв’язку», додані до неправильних варіантів, пояснюють, чому вони неправильні.

📖 [Специфікація Kramdown — питання вікторини на знання (зі зворотним зв'язком щодо кожного варіанта відповіді)](http://digital-docs.rpf-internal.org/docs/technology/codebases-and-products/raspberry-flavoured-markdown/specs/kramdown_rpf-legacy-spec#with-per-choice-feedback)

```markdown

Який режим виводу GPIO Raspberry Pi слід використовувати для ввімкнення та вимкнення світлодіода?


- (x) Вихід

- ( ) Вхід

  --- зворотний зв'язок ---
  Вхідний режим зчитує сигнал — він не може керувати таким компонентом, як світлодіод.
  --- /зворотній зв'язок ---

- ( ) ШІМ

  --- зворотний зв'язок ---
  ШІМ (широтно-імпульсна модуляція) керує яскравістю, а не простим увімкненням/вимкненням.
  --- /зворотній зв'язок ---


```

Який режим виводу GPIO на Raspberry Pi слід використовувати для ввімкнення та вимкнення світлодіода?

- (x) Вихід

- ( ) Вхід

  Вхідний режим зчитує сигнал — він не може керувати компонентом, таким як світлодіод.

- ( ) ШІМ

  ШІМ (широтно-імпульсна модуляція) керує яскравістю, а не простим увімкненням/вимкненням.
